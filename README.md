# DropdownMenu
Create a Dropdown Menu widget for Android. In order to create a Menubar on tablet, display menu item with dynamic width. Thanks to [dongjunkun/DropDownMenu](https://github.com/dongjunkun/DropDownMenu).

Acknowledge
====
Thanks to [dongjunkun/DropDownMenu](https://github.com/dongjunkun/DropDownMenu)

中文
----
创建一个多条件筛选下拉菜单，学习了[dongjunkun/DropDownMenu](https://github.com/dongjunkun/DropDownMenu)筛选菜单的实现，重构了代码，将菜单标题栏实现分离，并可以在平板等设备上任意位置放置菜单栏，设置下拉菜单的宽度，并设置全屏遮罩。实现了网页下拉菜单的效果。
